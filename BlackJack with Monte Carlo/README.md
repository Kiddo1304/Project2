# Solving Blackjack with Monte Carlo Methods

# Overview
This project implements reinforcement learning
methods to learn an optimal Blackjack strategy.

## Algorithms
- Monte Carlo Exploring Starts
- Monte Carlo Control (epsilon-greedy)

## Results
MC Control achieved a higher win rate (~42.9%) compared to Exploring Starts (~38.2%).

## How to Run
pip install -r requirements.txt  
python main.py

## Dependencies
- numpy
-matplotlib
